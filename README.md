# Profiles_API
User profiles API with Django Rest Framework

## Requirements
1)Vagrant
2)Virtual Box


### Start The Vagrant Machine

```
vagrant up
```

### SSH to the machine

```
vagrant ssh
```

### Start the virtual env

```
cd /vagrant && source ~/venv/bin/activate
```

### Start the development server

```
python3 manage.py runserver 0.0.0.0:8000
```
